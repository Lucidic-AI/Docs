# 🚀 Installation

The **Lucidic AI SDK** lets you record, evaluate, and debug your AI agents by connecting them to the Lucidic platform.

---

## 📦 Install the SDK

Our package is published on PyPI under the name `lucidicai`.

```bash
pip install lucidicai
```

We recommend installing it into a virtual environment or as part of your existing agent stack.

---

## 🧪 Minimum Requirements

- Python **3.8+**
- Compatible with all major LLM frameworks and agent libraries
- Works in local, server, or cloud environments

---

## ✅ Verify Installation

To confirm it’s installed correctly:

```bash
python -c "import lucidicai; print(lucidicai.__version__)"
```

You should see the SDK version printed without any errors.

---

## 🌐 Next Step: Authenticate

Once installed, head to the [Authentication guide](/getting-started/authentication) to connect your SDK to your Lucidic workspace.
