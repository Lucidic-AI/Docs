---
title: sessions.run
description: Run a function inside a new session context
---

## Overview

`sessions.run()` creates a new session, runs your function within that context, and then ends the session after the function returns.

## Syntax

```python
result = client.sessions.run(
  fn: Callable[..., T],
  session_name: Optional[str] = None,
  *args,
  **kwargs
) -> T
```

## Parameters

- `fn` (callable, required): Function to execute within the session.
- `session_name` (string, optional): Display name for the session.
- `*args`, `**kwargs`: Arguments forwarded to `fn`.

## Returns

Returns whatever `fn` returns.

## Example

```python
from lucidicai import LucidicAI

client = LucidicAI(api_key="...", providers=["openai"])

def work(x: int) -> int:
    return x * 2

result = client.sessions.run(work, session_name="wrapped", x=5)
```
