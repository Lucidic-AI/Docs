---
title: sessions.run_in
description: Run a function inside an existing session context
---

## Overview

`sessions.run_in()` binds the given session id, executes your function, and preserves the session (does not end it).

## Syntax

```python
result = client.sessions.run_in(
  session_id: str,
  fn: Callable[..., T],
  *args,
  **kwargs
) -> T
```

## Parameters

- `session_id` (string, required): Existing session id to bind.
- `fn` (callable, required): Function to execute within the session context.
- `*args`, `**kwargs`: Arguments forwarded to `fn`.

## Returns

Returns whatever `fn` returns.

## Example

```python
from lucidicai import LucidicAI

client = LucidicAI(api_key="...", providers=["openai"])

session_id = client.sessions.create(session_name="external", auto_end=False)

def do_work(x: int) -> int:
    return x + 10

result = client.sessions.run_in(session_id, do_work, x=7)
```
