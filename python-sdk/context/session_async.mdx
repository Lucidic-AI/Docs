## Overview

`sessions.acreate()` is the async counterpart to `sessions.create()`. It creates/binds a session and ends it when the async context exits.

## Syntax

```python
async with client.sessions.acreate(
  session_name: Optional[str] = None,
  session_id: Optional[str] = None,
  task: Optional[str] = None,
  experiment_id: Optional[str] = None,
  rubrics: Optional[list] = None,
  tags: Optional[list] = None,
  production_monitoring: Optional[bool] = False,
):
  ...
```

## Returns

A session context. Binds the active session for the duration of the async context.

## Example

```python
from lucidicai import LucidicAI

client = LucidicAI(api_key="...", providers=["openai"])

async def handle_request():
    async with client.sessions.acreate(session_name="async task"):
        await do_async_work()
# Session auto-ends when context exits
```
