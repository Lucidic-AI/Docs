## Overview

Async version of `sessions.bind()`. Binds an existing session for the duration of the async context, without ending it on exit.

## Syntax

```python
async with client.sessions.bind_async(session_id: str):
  ...
```

## Example

```python
from lucidicai import LucidicAI

client = LucidicAI(api_key="...", providers=["openai"])

session_id = client.sessions.create(session_name="external", auto_end=False)

async def handle():
    async with client.sessions.bind_async(session_id):
        await do_more_async_work()
```
