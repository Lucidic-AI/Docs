---
title: sessions.bind
description: Bind an existing session id for a block of work (no end on exit)
---

## Overview

`sessions.bind()` attaches an existing session to the current context for the duration of the `with` block. The session is not ended on exit â€” end it explicitly later.

## Syntax

```python
with client.sessions.bind(session_id: str):
  ...
```

## Parameters

- `session_id` (string, required): An existing session id to bind.

## Returns

None. Sets the active session for the context.

## Example

```python
from lucidicai import LucidicAI

client = LucidicAI(api_key="...", providers=["openai"])

session_id = client.sessions.create(session_name="request-123", auto_end=False)

with client.sessions.bind(session_id):
    do_work()

client.sessions.end()
```
