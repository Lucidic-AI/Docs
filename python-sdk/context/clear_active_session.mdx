---

## Overview

Clears any active session id from the current context. Use after manual `set_active_session(...)` or when leaving a custom context boundary.

## Syntax

```python
lai.clear_active_session() -> None
```

## Example

```python
import lucidicai as lai

sid = lai.init(session_name="manual", providers=["openai"], auto_end=True)
lai.set_active_session(sid)
# ... work ...
lai.clear_active_session()
lai.end_session()
```
