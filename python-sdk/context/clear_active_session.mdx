title: sessions.clear_active
description: Clear the active session id for the current context
---

## Overview

Clears any active session id from the current context. Use after manual `sessions.set_active(...)` or when leaving a custom context boundary.

## Syntax

```python
client.sessions.clear_active() -> None
```

## Example

```python
from lucidicai import LucidicAI

client = LucidicAI(api_key="...", providers=["openai"])

session_id = client.sessions.create(session_name="manual", auto_end=False)
client.sessions.set_active(session_id)
# ... work ...
client.sessions.clear_active()
client.sessions.end()
```
