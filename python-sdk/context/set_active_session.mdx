## Overview

Manually sets the active session id for the current thread/async context. Prefer the higher-level context managers for most cases.

## Syntax

```python
client.sessions.set_active(session_id: Optional[str]) -> None
```

## Parameters

- `session_id` (string | None): Session to set as active. Pass `None` to clear.

## Example

```python
from lucidicai import LucidicAI

client = LucidicAI(api_key="...", providers=["openai"])

session_id = client.sessions.create(session_name="manual", auto_end=False)
client.sessions.set_active(session_id)
# ... work ...
client.sessions.clear_active()
client.sessions.end()
```
