## Overview

The `reset_sdk` function is deprecated and retained only for backward compatibility. It no-ops in current versions. Prefer ending sessions with `end_session()` and allowing automatic cleanup.

## Syntax

```python
lai.reset_sdk() -> None  # Deprecated, no-op
```

## Parameters

This function takes no parameters.

## Returns

This function returns `None`.

## Examples

### Basic Usage (Deprecated)

```python
import lucidicai as lai

# Start a session
lai.init(session_name="First Session", providers=["openai"])

# Do some work...
# ...

# Reset everything
# Deprecated: no-op
lai.reset_sdk()

# Now you can start fresh
lai.init(session_name="Second Session", providers=["anthropic"])
```

## Notes

- Deprecated: function is a no-op in current versions
- Use `end_session()` for proper cleanup

## Behavior Details

When called, `reset_sdk` performs no action.

## Best Practices

1. Prefer explicit `end_session()` in tests and workflows
2. Ensure sessions are ended before changing contexts
